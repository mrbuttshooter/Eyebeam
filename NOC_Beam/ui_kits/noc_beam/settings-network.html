Transport
Protocol network.transport.proto
App-wide default. Each account can override. TLS is recommended for any account that crosses the public internet.
Listen port network.transport.port
auto-fallback to 5060/UDP
Single port that handles all transports. Auto-fallback retries on 5060/UDP if the TLS bind fails (corp firewall etc.).
Bind interface network.transport.bind_addr
Restrict to a single NIC if the host is multi-homed (Ethernet + Wi-Fi). 0.0.0.0 = listen everywhere.
NAT traversal
STUN server network.stun.server
Used by the NAT probe to discover your public address. Multiple servers can be comma-separated. Internal STUN is preferred — public servers (e.g. stun.l.google.com) leak topology.
TURN server (relay) network.turn.server
off — full cone, not needed
Last-resort media relay when peer-to-peer can't traverse. Currently off because the probe detected full-cone NAT. Enable for symmetric NATs.
ICE network.ice.enabled
Interactive Connectivity Establishment — discovers the best media path. Aggressive for lab; Standard for production trunks where SBC handles relay.
Keep-alive network.keepalive.interval_s
seconds
Empty SIP packets that keep NAT pinholes open. 30 s is safe for almost any consumer router; bump to 15 s for aggressive timeouts on hotel/coffee-shop Wi-Fi.
TLS
Server certificate network.tls.cert_path
noc-internal.crt · CN=*.noc.internal · expires 2027-04-18 Browse…
Presented to peers when accepting inbound TLS connections. Watch the expiry date — the app shows a banner at 30 days remaining.
Trust store network.tls.ca_bundle
Windows uses the system Certificate Store. Custom when you have a private CA. Both = union (rare; useful during CA rotation).
Verify peer network.tls.verify_peer
Strict is the only acceptable setting on a registered production endpoint. Off is fine for poking at a lab SBC but never reaches saved accounts.
Min TLS version network.tls.min_version
TLS 1.0/1.1 are not offered. 1.3 recommended; downgrade to 1.2 only when peering with legacy SBCs that haven't been patched.
