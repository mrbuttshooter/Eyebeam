Audio

Devices, levels and ringtone. Per-account overrides live in Accounts.
Devices
Input device audio.input.device
Device used for outgoing audio. Speak now — meter should peak around 6 bars; if it clips, lower input gain below.
Input gain audio.input.gain_db
+2 dB
Applied before sending to the codec. Range −12 dB to +12 dB.
Output device audio.output.device
Where call audio plays. Ringtone uses a separate output, set below.
Ringtone output audio.ringtone.device
Plays incoming-call ringtone. Use a different device when wearing headphones at low volume.
Processing
Echo cancellation audio.aec
on · WebRTC AEC3
Removes the far-end voice from your microphone signal. Disable only when using a hardware AEC (USB headset, conference phone).
Noise suppression audio.ns
on · medium
Suppresses fan, keyboard, room tone. High can soften consonants — leave on medium unless the room is loud.
Automatic gain control audio.agc
off
Adjusts mic gain to match a target loudness. Off by default — most decent mics don't need it and AGC fights with manual gain.
Behaviour
Mute on join audio.mute_on_answer
off
Start every accepted call with your mic muted. Useful for bridges where you join silently first.
Push-to-talk audio.ptt_keybind
Hold this key to unmute. Released → muted. Captured globally (works while another window has focus).
