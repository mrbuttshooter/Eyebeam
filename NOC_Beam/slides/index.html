Q3 retrospective/03 · what changed 05 / 08
$ git log --since=Q3 --oneline

Five changes that moved the line.

  • Rolled NOC_Beam 0.1.0 to all 12 NOC desks — Eyebeam decommissioned (closes #218)
  • Added SHA-256 digest to the SIP auth path; gated the SBC rotation behind it
  • Standardised on opus/48000/2 as the default codec; relegated G.711 to fallback
  • Synthetic-call canary now runs every 90s across every trunk; PagerDuty page on 3 consecutive 4xx
  • Trace export is wired into the on-call ticket template — no more "can you DM me the log?"

Q3 retrospective/04 · why we left Eyebeam 06 / 08
$ diff eyebeam noc_beam

What we left, what we built.

BeforeEyebeam 1.5

  • Closed-source, last release 2012
  • No SIP trace — opaque failure modes
  • One account at a time; switching loses state
  • TLS but no SRTP; no per-account toggles
  • Windows-only · installer ships with toolbar adware
  • Licence cost: $$$ per seat, per year

AfterNOC_Beam 0.1

  • PySide6 + PJSIP, owned by us, audited quarterly
  • Live SIP trace, RX/TX coloured, filterable
  • Multi-account; switching is a combo, not a relaunch
  • TLS + SRTP optional/mandatory per account
  • Single-file .exe via PyInstaller; no installer
  • Free, internal, repo-driven release process
Migration: 100% · Q2 → Q3 NOC_Beam
